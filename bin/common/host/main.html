Fast Internet Radio Engine

Firecast is a media server that can be used to create an network streaming radio station. The server broadcasts streams from sources to clients (listeners).

As sources can be used: ices0, ices2, Winamp (windows) with SHOUTcast DSP Plug-in, PM123 (OS/2, eComStation) with fires-PM123 filter plug-in, shout utility. For listening a stream any suitable player can be chosen (OS/2, eComStation: PM123, Z!; windows: winamp; e.t.c.).

All management is made through a web-interface. Administrator do not have necessity to edit any configuration files (while it has an access to the web-interface).

For correct representation of the information it is required to use internet-browser with XML/XSLT and JavaScript support. For instance:

  • Mozilla
  • Firefox
  • SeaMonkey
  • MS Internet Explorer 6.0
