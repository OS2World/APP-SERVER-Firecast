Что такое Firecast?

Firecast представляет собой сервер аудиопотоков форматов Ogg Vorbis или MP3. Слушатели подключаются к серверу, используя различное мультимедийное программное обеспечение. Для воспроизведения аудиопотоков по сети могут быть использованы многие популярные аудио-плееры.

Firecast является бесплатным программным обеспечением с открытым исходным кодом. Могут использоваться сопутствующие продукты как с открытым исходным кодом (такие как fires-PM123 filter plug-in для плеера PM123, ices0/ices2, shout из проекта icecast.org), так и коммерческие продукты (например, бесплатный SHOUTcast DSP Plug-in для Winamp).

Для передачи данных от сервера к клиентам используется http-протокол и, обычно, tcp/ip порт 8000. Для работы с "источниками" (см. далее) в некоторых случаях используется дополнительный порт 8001. Эти tcp/ip порты должны быть доступны сетевым клиентам. Данные значения являются общепринятыми, но не обязательными (могут изменяться).
